# SafeFind - Modern Dashboard UI

A clean, modern, responsive web dashboard UI for a women safety and missing person identification platform.

## 🎨 Design Features

- **Modern Card-Based Interface**: Inspired by mobile UI patterns
- **Gradient Header**: Blue to purple gradient with subtle pattern
- **Soft Shadows**: Neumorphism-inspired card design
- **Responsive Grid**: 2 columns (mobile) → 3 columns (tablet) → 4 columns (desktop)
- **Icons Above Text**: Each feature card has icon on top
- **Smooth Transitions**: Hover effects and page transitions

## 🛠️ Tech Stack

- React.js 18 with Vite
- Lucide React for icons
- CSS (custom styling, no Tailwind)
- React Router v6

## 📦 Installation

```bash
cd safefind-frontend
npm install
npm run dev
```

## 📱 Pages Included

1. **Home** - Dashboard with stats and feature cards
2. **Report Missing** - Form to report missing person
3. **Upload Found** - Image upload for face matching
4. **Cases** - Search and filter all cases
5. **Case Detail** - Individual case view with sightings
6. **Community** - Community reports and sightings
7. **About** - Project information

## 🎯 Features

- 8 feature cards on dashboard
- Quick action buttons
- Recent cases list
- Stats counters
- Filter and search functionality
- Form validation
- Loading states
- Alert messages
- Responsive design

## 📱 Responsive Breakpoints

- Mobile: < 600px (2 columns)
- Tablet: 600px - 900px (2 columns)
- Desktop: 900px - 1200px (3 columns)
- Large: > 1200px (4 columns)
